Игра ультиматум для группы

Один файл без сервера. Подходит для быстрого запуска на занятии: случайно делит студентов на роли, собирает ответы и считает, состоялась ли сделка по среднему.

Есть два режима: агрегированный по среднему и парный. Можно открыть локально в браузере как обычный index.html.
